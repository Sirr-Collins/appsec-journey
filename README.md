
# My AppSec Journey

Documenting my progress, custom tools, and lab exercises as I navigate Application Security.

## 🛠️ Custom Tools
* [AETHER: Tactical Port Scout](./port_scanner) - A multi-threaded TCP port scanner built in Python.
* [AETHER: Custom Log Analyzer](./log_analyzer) - A python script to parse and extract threat intelligence from Apache/Nginx web logs.
* [AETHER: JWT Claims Extractor](./jwt_decoder) - A dependency-free, offline JSON Web Token inspector for auditing session claims.
* [AETHER: HTTP Request Forger](./http_forger) - An active engagement tool for spoofing HTTP headers and injecting custom cookies via the `requests` library.
